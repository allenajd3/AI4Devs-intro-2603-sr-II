#Objetivo
Construir una página web que permita introducir una cadena de texto y al pulsar un botón muestre debajo la misma cadena en orden inverso

Debe de tener un botón "Reverse" que es el que dispara la acción
Debe de incluir un Botón "Copy" que permita copiar el resultado (texto invertido)
#Tecnologías
HTML y JAVASCRIPT. No necesito que importes librerías externas, unicamente utiliza como base el HTML adjunto y el archivo JS para que añadas en él la lógica que necesites

#Estilo
Estilo simple y sin florituras, añade algún color a los botones pero nada complejo. Adjunto imagen de ejemplo de resultado

#Salida
Indicame por el chat el contenido final de ambos ficheros (index.html y script.js)
